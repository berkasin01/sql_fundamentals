# SQL Fundamentals

Practicing SQL queries on a sleep health dataset using SQLite and pandas in Jupyter Notebook.

## Dataset

Sleep Health and Lifestyle dataset from Kaggle. 374 rows covering sleep duration, quality, stress levels, BMI, occupation, physical activity and more.

## Questions Explored

1. Which occupation gets the most sleep on average?
2. Does stress level affect sleep quality?
3. Who has high stress but poor sleep?
4. Average heart rate by BMI category
5. How many people per occupation?
6. Who exercises a lot but still sleeps badly?
7. Average sleep duration by gender
8. Which BMI category has the highest stress?
9. Does physical activity lower heart rate?
10. Most common sleep disorder
11. Who sleeps 8+ hours but still has low sleep quality?
12. Average physical activity by occupation

## Stack

Python, SQLite, pandas, Jupyter Notebook
